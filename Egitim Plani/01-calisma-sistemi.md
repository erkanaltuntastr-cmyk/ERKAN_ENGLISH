# 01 - Calisma Sistemi

This plan is designed for continuing through ChatGPT, Codex, or another AI assistant that can read this repository.

## Weekly Cycle

1. Review last week's progress notes.
2. Select one source or topic.
3. Create original study notes.
4. Complete exercises or a mini test.
5. Correct mistakes and add them to the mistakes log.
6. Write a short reflection in the weekly log.

## Output Types

Use these formats in `Olusturulan icerikler`:

| Type | Folder | Description |
| --- | --- | --- |
| Study notes | `notes/` | Original explanations and examples |
| Vocabulary | `vocabulary/` | Word lists with British pronunciation focus and original sentences |
| Grammar | `grammar/` | Rules, patterns, and controlled practice |
| Writing | `writing/` | Prompts, drafts, corrections, improved versions |
| Speaking | `speaking/` | Speaking prompts, model answers, pronunciation notes |
| Listening | `listening/` | Listening tasks based on public audio/video links |
| Exams | `exams/` | Original quizzes, mock exams, answer keys |

## Source Use Rule

For each generated item, include:

- Source ID or public source link
- Topic
- CEFR level
- Date
- Whether the output is original notes, quiz, exam, or correction

Do not include long copied text from the source.
