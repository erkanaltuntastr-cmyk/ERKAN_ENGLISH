# 03 - Just Enough English Grammar Sprint

Source: BOOK-08, `Just Enough English Grammar Illustrated gabriele stobbe.pdf`

Decision: use this book before the full level diagnostic as a short grammar reset.

This is a sensible first step if it is treated as an active sprint, not passive reading. The book is visual and compact, so it can expose basic gaps quickly and build confidence before the broader CEFR diagnostic.

## Why This Works

- It covers the foundations: nouns, adjectives, pronouns, verbs, adverbs, prepositions, conjunctions, and interjections.
- It is short enough to finish.
- It can reveal repeated grammar weaknesses before the placement test.
- It is useful for building a shared grammar vocabulary between Erkan and the AI assistant.

## Risk

This book may be below Erkan's real level in some areas. If every exercise feels easy, do not slow down. Convert it into a fast review and move to the placement test.

## Rule

Do not just read. Every session must produce one of these:

- a short original study note
- a self-made example list
- a mini quiz
- an error log entry
- a short writing/speaking task using the target grammar

## Target Pace

Recommended duration: 10 study sessions.

Session length: 35-60 minutes.

If studying daily, finish in about 2 weeks. If studying 3 times per week, finish in about 4 weeks.

## Stop Or Continue Rule

After Session 3, decide:

- If accuracy is 85% or higher and the material feels easy, speed up and finish the rest as review.
- If accuracy is below 70%, slow down and create extra practice before moving on.
- If accuracy is between 70% and 85%, continue at the normal pace.

After Session 10, run the full level diagnostic from:

`Egitim Plani/02-seviye-tespit-hazirlik.md`

## First Task

Session 1 should produce:

1. A short note: "What I already know about English grammar"
2. A short note: "What I usually confuse"
3. Five original English sentences about work, AI, and learning
4. A correction pass by the AI assistant
