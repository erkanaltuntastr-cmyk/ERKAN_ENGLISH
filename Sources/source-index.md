# ERKAN_ENGLISH Source Index

This repository uses owned study materials and public learning resources as references.
Do not copy full chapters, pages, scans, or long excerpts into tracked repo files.

Allowed use:
- Create original study notes based on concepts from the sources.
- Create original exercises, quizzes, mock exams, answer keys, and feedback.
- Record page references, topic names, CEFR levels, and short paraphrased summaries.
- Link to the original source location.

Not allowed in tracked GitHub files:
- Copy full PDF text, pages, tables, answer keys, or images.
- Reproduce copyrighted examples at length.
- Publish substantial source-derived material that could replace the original books.
