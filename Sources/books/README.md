# Books Reference Guide

This folder is a local reference library. PDF files in this folder are intentionally ignored by Git and should not be published to GitHub.

Use the books actively when preparing Erkan's English learning plan, study notes, exercises, quizzes, and mock exams. The expected output is original teaching material, not copied book pages.

## Usage Rules

- Read the PDFs as reference material when needed.
- Use book concepts, topic order, page references, and exercise types to design original lessons.
- Direct excerpts may be used only when truly useful for private study, correction, or comparing an answer with the source.
- Do not paste long sections, full pages, full answer keys, scans, or images into tracked repo files.
- In generated notes, prefer paraphrased explanations and original examples.
- When a lesson is based on a book, include the local source ID and topic.
- For GitHub publishing, keep only catalogues, plans, notes, generated exercises, and progress logs.

## Classification

### Core Grammar

Use these for the main grammar curriculum:

- BOOK-01: primary path for intermediate grammar.
- BOOK-05: reference for grammar explanations and edge cases.
- BOOK-06: practice source for grammar reinforcement.
- BOOK-08: quick visual explanation source for weak areas.

### Secondary Grammar

Use these only when extra explanations, contrast, or more practice is needed:

- BOOK-03: older intermediate grammar edition.
- BOOK-07: comprehensive grammar, useful but not British-first.

### Vocabulary and Phrasal Verbs

Use these for vocabulary modules, speaking fluency, and natural English:

- BOOK-09
- BOOK-10

## Learning Material Output Format

Every generated file should include this header:

```text
CEFR:
Skill:
Source:
Topic:
Date:
Output type:
```

Allowed output types:

- Study note
- Mini lesson
- Controlled practice
- Review quiz
- Mock exam
- Speaking prompt
- Writing prompt
- Error correction set

## Recommended Workflow

1. Identify the target CEFR level and weak skill area.
2. Select one active book as the main reference.
3. Read only the needed topic or pages.
4. Produce original explanations and examples.
5. Add practice questions with an answer key.
6. Record repeated mistakes in `Gelisim notlari/mistakes-log.md`.

## Level Diagnostic Use

For seviye tespit, do not rely only on book placement material. Use a combination of:

- A short grammar diagnostic based on BOOK-01 and BOOK-05 topics.
- A vocabulary/phrasal verb diagnostic based on BOOK-09 and BOOK-10.
- A writing sample.
- A speaking transcript or recording summary.
- Optional external CEFR tests from British Council, Cambridge, or Oxford.
